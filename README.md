# Equilibragem

Backend API for the Equilibragem financial control application.

## Tech Stack
- **Runtime**: [Bun](https://bun.sh)
- **Framework**: [ElysiaJS](https://elysiajs.com)
- **Database**: PostgreSQL
- **ORM**: Drizzle ORM

## Getting Started

To install dependencies:
```bash
bun install
```

To start the development server:
```bash
bun run dev
```

The server will be running at http://localhost:3000

## Formatting

This project uses [Biome](https://biomejs.dev) for code formatting and import organization.

- Fix formatting issues: `bun run format`
- Check formatting without writing changes: `bun run format:check`
